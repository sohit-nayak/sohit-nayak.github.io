Recommendation System for Fashion website ASOS

Project information

  • Category: Deep Learning
  • Project date: 10 Jan, 2023
  • Project URL: Github Link

The following project attempts in building a Recommendation System for a fashion e-commerce website, ASOS. Recommender systems are algorithms aimed at suggesting relevant items to users (items being movies to watch, text to read, products to buy or anything else depending on industries). Collaborative Filtering Method has been used which solely is based on the past interactions recorded between users and items in order to produce new recommendations. Then, the main idea that rules collaborative methods is that these past user-item interactions are sufficient to detect similar users and/or similar items and make predictions based on these estimated proximities.
