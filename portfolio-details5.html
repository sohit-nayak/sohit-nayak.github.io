Image Prediction using Convolutional Neural Networks (Tensorflow)

Project information

  • Category: Machine Learning
  • Project date: 22 November, 2022
  • Project URL: Github Repository

This project attempts to predict cats from dogs. The dataset is a set of more than 30,000 images of cats and dogs. We attempt to do this by building a multi-layered Convolutional Neural Network in Keras, Tensorflow. The architecture of the network is: Conv2D (64 neurons) -> MaxPooling2D -> Conv2D (64 neurons) -> MaxPooling2D -> Dense (64 neurons) -> Output. Binary Cross Entropy is the loss funtion and we take a look at the accuracy of the predictions. We then analyze the results in TensorBoard. The blue line in the results here represents the validation set and the orange line is the training set.
